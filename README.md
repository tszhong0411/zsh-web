## Features

You can run commands like `ls`, `mkdir`, `rm` on web.

## Installation

1. Clone the repository.

```shell
git clone https://github.com/tszhong0411/zsh-on-web.git
```

2. Install dependencies.

```shell
pnpm install
```

3. Start the server.

```shell
pnpm dev
```
